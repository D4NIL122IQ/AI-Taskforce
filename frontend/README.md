# AI Taskforce — Frontend

Interface web du projet AI Taskforce, construite avec React et Vite.

## Stack

- React 19
- Vite
- Tailwind CSS
- React Router DOM
- Axios

## Prérequis

- Node.js >= 18
- npm

## Installation

```bash
npm install
```

## Lancer en développement

```bash
npm run dev
```

L'application est disponible sur `http://localhost:5173`
